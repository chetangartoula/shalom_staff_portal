# **App Name**: CostMatrix

## Core Features:

- Editable Cost Table: Interactive table with columns for Serial No, Description, Rate, Times, and Total (auto-calculated).
- Add/Delete Rows: Functionality to add new rows and delete existing rows from the cost table.
- Reactive Calculations: Real-time updates of Total and Subtotal as values change in the table.
- Summary Card: Display Subtotal, Discount, and Net Total at the bottom of the page, in a sticky footer.
- Local Storage Saving: Automatically save calculations to local storage for persistent data.
- Export to PDF: Generates a downloadable PDF report of the cost calculation.
- Export to Excel: Generates an Excel file of the cost calculation for offline use and sharing.

## Style Guidelines:

- Primary color: Light teal (#90EE90) to create a friendly and inviting interface.
- Background color: Very light green (#F0FFF0), providing a subtle, clean backdrop.
- Accent color: Darker green (#3CB371) to highlight interactive elements.
- Font: 'Inter' sans-serif for a modern, clean, and highly readable experience, suitable for both headers and body text.
- Mobile-first design, ensuring responsiveness across all devices.
- Card-based layout for sections, with consistent spacing for a clean, organized appearance.
- Sticky footer for the Summary Card, ensuring it's always visible.
- Subtle transition animations on data changes to enhance user experience.
